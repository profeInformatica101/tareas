Seleccionando y utilizando las estructuras de programación

Instrucciones: Realiza un fork del siguiente repositorio y asegúrate de mantener el mismo nombre (coches-thymeleaf): https://github.com/profeinformatica101/coches-thymeleaf.
  • Después del fork, tu repositorio debe estar disponible en la URL: https://github.com/tu_usuario/coches-thymeleaf.
  • Clona tu repositorio localmente y realiza las modificaciones necesarias.
  • Sube los cambios al repositorio y asegúrate de documentar el trabajo.

  • Se han utilizado mecanismos de decisión en la creación de bloques de sentencias.
  • Se han utilizado bucles y se ha verificado su funcionamiento.
  • Se han utilizado matrices (arrays) para almacenar y recuperar conjuntos de datos.
  • Se han creado y utilizado funciones.
  • Se han utilizado formularios web para interactuar con el usuario del navegador web.
  • Se han empleado métodos para recuperar la información introducida en el formulario.
  • Se han añadido comentarios al código.

Cuando el usuario acceda a http://localhost:8183/coches, se deberá mostrar una tabla con todos los coches.

  • Desarrollar una función en ControladorCoche para manejar las peticiones GET en la ruta /coches.
  • Crear la página coches.html usando etiquetas de Thymeleaf.

Ampliar la funcionalidad del ejercicio anterior para incluir un selector que permita filtrar coches por fuente de energía.

  • Mostrar los coches de la fuente seleccionada, si existen.
  • Mostrar un mensaje si no existen coches para la fuente seleccionada.

Añadir a la página principal de coches un botón que permita acceder al formulario. El botón debe estar incluido en fragmentos.html.

Cuando el usuario acceda a http://localhost:8183/formulario, deberá ver un formulario para agregar nuevos coches.

  • Crear una función en ControladorCoche para manejar peticiones GET en la ruta /formulario.
  • Crear una función en ControladorCoche para manejar peticiones POST en la ruta /nuevo_coche.
  • Validar los campos y mostrar mensajes de error si es necesario.

Implementar la funcionalidad para que las cabeceras de la lista de coches se muestren en el idioma especificado mediante el parámetro ?lang en la URL.

RECUERDA: Es obligatorio realizar comentarios en el código.
