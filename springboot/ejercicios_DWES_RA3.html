Ejercicios usando Spring Boot

Sigue los pasos detallados para trabajar en el repositorio.

Visita el repositorio principal: https://github.com/profeInformatica101/ejercicios_spring_boot y realiza un fork para crear una copia en tu cuenta de GitHub.

Clona tu repositorio forkeado en tu entorno local. Ejecuta el siguiente comando en tu terminal:

git clone <URL-del-repositorio>

Reemplaza <URL-del-repositorio> con la URL de tu fork.

Para trabajar en cada ejercicio, crea una rama nueva utilizando el siguiente comando:

git checkout -b ejercicio-<número>

Ejemplo: git checkout -b ejercicio-1.

Realiza los cambios correspondientes en la rama de cada ejercicio.

Después de realizar los cambios en la rama de un ejercicio, confirma los cambios con:

git add .
git commit -m "Solución del ejercicio <número>"

Después de completar un ejercicio, haz un merge de la rama en main:

git checkout main
git merge ejercicio-<número>

Finalmente, sube los cambios al repositorio remoto:

git push origin main
