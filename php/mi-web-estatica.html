Publicación de Página Web Estática en GitHub Pages

Sigue estos pasos para crear y publicar tu página web estática en GitHub Pages.

Ve a tu cuenta de GitHub y crea un nuevo repositorio público llamado mi-web-estatica.

Clona el repositorio en tu entorno local utilizando el siguiente comando:

git clone https://github.com//mi-web-estatica

Tu URL pública será:

  • Mi URL pública: https://profeinformatica101.github.io/mi-web-estatica/
  • Tu URL pública: https://<tu-usuario>.github.io/mi-web-estatica/

Desarrolla un diseño básico en HTML y CSS. Puedes utilizar cualquier framework como Bootstrap para simplificar el diseño.

Haz commit de tus cambios y súbelos al repositorio en GitHub con los siguientes comandos:

git add .
git commit -m "Subida inicial del diseño de mi web estática"
git push origin main

En tu repositorio en GitHub, ve a Settings > Pages. Configura la publicación de GitHub Pages seleccionando la rama main como la fuente y guarda los cambios.

Tu sitio web debería estar accesible en la URL generada por GitHub Pages y mostrar correctamente el contenido HTML que has creado.

Enlace de tu sitio web: https://<tu-usuario>.github.io/mi-web-estatica/
