Proyecto CRUD de Libros

Sigue los pasos detallados a continuación para completar el proyecto de CRUD de libros en tu entorno de desarrollo local.

Ve al repositorio: https://github.com/profeInformatica101/crud_php_example y haz clic en Fork para crear una copia en tu cuenta de GitHub.

En tu repositorio forkeado, haz clic en Code y copia la URL. Luego, clona el repositorio en tu entorno local:

git clone <URL-del-repositorio>

Crea una nueva rama llamada crud_libro y cambia a esta rama:

git branch crud_libro
git checkout crud_libro

Verifica las ramas existentes con git branch -a.

Ejecuta el archivo SQL en /crud_php_example/bbdd/tienda.sql utilizando PHPMyAdmin:

  • Abre PHPMyAdmin y selecciona o crea una base de datos (por ejemplo, tienda).
  • Importa el archivo tienda.sql desde la sección de Importar.

Crea una nueva rama para trabajar en el CRUD de libros:

git branch libro
git checkout libro

Modifica los archivos .php del CRUD actual para que trabajen con una tabla llamada libros en la base de datos.

Para subir los cambios de manera regular, utiliza los siguientes comandos:

git add .
git commit -m "Descripción adecuada del cambio"
git push origin libro

Es recomendable utilizar un mensaje descriptivo para cada cambio que realices.

Inicia los servicios nuevamente y accede a la aplicación en el navegador para verificar que las operaciones de CRUD de libros funcionan correctamente.
