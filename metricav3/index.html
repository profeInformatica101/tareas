📌 Factor de Calidad 📏 Métrica / Indicador ✅ Excelente (10-9) 🟡 Aceptable (8-6) ❌ Mejorable (5 o menos)
🎯 Funcionalidad Cumplimiento de requisitos funcionales definidos en ASI Todos los requisitos cubiertos, sin errores Faltan detalles menores o errores aislados Funcionalidades incompletas o fallidas
🔒 Fiabilidad Pruebas y estabilidad del sistema Sin errores, manejo de excepciones correcto Errores menores, comportamiento predecible Errores graves, mal manejo de excepciones
🧱 Diseño lógico Diagrama de clases en Umbrello Completo, relaciones correctas y claras Algunas omisiones o errores leves Falta estructura o coherencia
🗄️ Acceso a datos Integración Java + SQLite vía JDBC CRUD funcional y probado en DBeaver CRUD parcial o errores menores No hay conexión funcional o errores graves
🧪 Verificabilidad Pruebas unitarias automatizadas (JUnit5) con cobertura ≥ 50% JUnit con mínimo 50% de cobertura, uso de dobles de prueba Pruebas básicas, automatización parcial, cobertura baja Sin pruebas o sin automatización
📚 Documentación README, Javadoc, comentarios Documentación clara y completa Documentación funcional pero breve Documentación incompleta o inexistente
🔁 Control de versiones Uso de Git y GitHub Repositorios con commits claros y ramas Uso básico de Git con seguimiento razonable Repositorios sin estructura o mal uso
🧹 Optimización Refactorización y organización del código Código limpio, modular y mantenible Organización aceptable, con mejoras posibles Código confuso o duplicado
🎤 Presentación final Exposición del proyecto Claridad, dominio técnico y visualización eficaz Explicación aceptable con dudas menores Poca claridad o desorganización en la exposición

📊 Evaluación total: hasta 90 puntos (rúbrica técnica) + 10 puntos (presentación oral).
📎 Entrega final: Repositorio GitHub + PDF de memoria + demo funcional + entregas según MÉTRICA v3 e ISO 12207.
📦 Maven: uso obligatorio de `pom.xml` para la gestión de dependencias.
🧪 Pruebas: obligatorio usar JUnit 5 con al menos un 50% de cobertura en clases funcionales clave.
