Apache Maven Wrapper

Simplifica la gestión de versiones de Maven en tus proyectos.

Documentación Oficial

¿Qué es Maven Wrapper?

El Maven Wrapper es una herramienta que garantiza que los usuarios de tu proyecto tengan la versión correcta de Maven para ejecutar builds, sin necesidad de instalarla manualmente. Es especialmente útil para proyectos que requieren una versión específica de Maven.

¿Por qué usar Maven Wrapper?

Aunque Maven es estable y está ampliamente disponible, el Wrapper simplifica la configuración al descargar e instalar automáticamente la versión correcta de Maven. Esto evita problemas de compatibilidad y asegura builds consistentes.

Cómo configurar Maven Wrapper

Para agregar el Maven Wrapper a tu proyecto, ejecuta el siguiente comando: 

mvn wrapper:wrapper

Esto generará los archivos necesarios, incluyendo los scripts mvnw (para Unix) y mvnw.cmd (para Windows).

Uso del Wrapper

En lugar de usar mvn, los usuarios pueden ejecutar: 

./mvnw clean install

O en Windows: 

mvnw.cmd clean install

Si la versión de Maven especificada no está instalada, el Wrapper la descargará automáticamente.

Sistemas compatibles

  • Linux (Ubuntu, CentOS, etc.)
  • macOS
  • Windows
  • Solaris (10 y 11)
  • BSD

Modo detallado

Para activar el modo detallado, configura la variable de entorno: 

export MVNW_VERBOSE=true

Descarga con autenticación

Si tu repositorio requiere autenticación, usa las variables de entorno: 

export MVNW_USERNAME=usuario
export MVNW_PASSWORD=contraseña

Verificación de integridad

Para evitar descargas corruptas, puedes especificar sumas de comprobación SHA-256 en maven-wrapper.properties: 

wrapperSha256Sum=abc123...
distributionSha256Sum=def456...
