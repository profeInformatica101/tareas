Implementación Colaborativa de Figuras Geométricas

  • Seleccionen un líder de equipo, quien será responsable de realizar el fork inicial del repositorio base: Repositorio Base.
  • El líder deberá clonar el repositorio forkeado y añadir a los demás integrantes del equipo como colaboradores.
  • Cada integrante trabajará en su propia rama de feature correspondiente a la figura asignada (por ejemplo: feature/circulo, feature/cuadrado).

  • La clase abstracta FiguraGeometrica se encuentra incluida en el proyecto base.
  • Cada clase deberá incluir:
    • Propiedades específicas de la figura.
    • Métodos necesarios para calcular áreas y perímetros.
    • Uso de etiquetas de Javadoc para documentar cada método y propiedad de la clase.
  • Implementa pruebas unitarias para validar el correcto funcionamiento de cada figura en la carpeta src/test/java/com/endes/figuras.

  • Cada integrante trabajará en una rama de feature correspondiente a su figura.
  • Pasos para realizar un pull request:
    1. Sube los cambios a tu rama de feature en el repositorio remoto (git push).
    2. En el repositorio en GitHub, selecciona tu rama y haz clic en "Compare & pull request".
    3. Rellena la descripción del pull request, explicando qué cambios se realizaron.
    4. Solicita la revisión del líder o de un compañero.
    5. Una vez aprobado, realiza el merge a la rama principal (main o master).
  • Comandos básicos de Git: 
    # Clonar el repositorio
$ git clone 

# Crear una nueva rama
$ git checkout -b feature/

# Añadir cambios al área de preparación
$ git add .

# Confirmar los cambios
$ git commit -m "Mensaje descriptivo"

# Subir la rama al repositorio remoto
$ git push origin feature/

  • Genera la documentación del proyecto con Javadoc y guárdala en la carpeta docs.
  • Incluye un archivo README.md con:
    • Una breve descripción del proyecto.
    • Listado de integrantes del equipo y las figuras asignadas.
    • Instrucciones para compilar, ejecutar y probar el código.

  • Una vez que el proyecto esté completo, cada integrante deberá realizar un fork del repositorio final del equipo y mantenerlo en su cuenta personal de GitHub con la ruta: https://github.com/<tu_usuario_github>/figuras_geometricas_base.
  • En Moodle, entrega el enlace a tu repositorio personal con el proyecto completo y funcional.
