Utilizando las clases Persona y LlenarPersonas

Código Java para las tareas
Persona.java 
                    public class Persona {
                        private String nombre;
                        private int edad;
                        
                        public Persona(String nombre,int edad)	{
                          this.nombre=nombre;
                          this.edad=edad;	
                         }
                        
                        public Persona() {
                          this.nombre=null;	  
                         }	
                        public void setNombre(String nom){nombre=nom;}
                        public void setEdad(int ed){edad=ed;}
                        
                        public String getNombre(){return nombre;}
                        public int getEdad(){return edad;}	
                    }//fin Persona
}

Coloca puntos de ruptura en los siguientes lugares:

  • Línea donde se instancia un objeto Persona dentro del bucle for en LlenarPersonas.
  • Línea donde se añade el objeto Persona a la lista lista.
  • Bucle for que muestra los elementos de la lista en consola.

Durante la depuración:

  • Inspecciona las variables nombres, edades, y lista.
  • Evalúa expresiones como nombres[i], edades[i], y lista.size() en el panel de Expresiones.

  • Utiliza Step Into para analizar cómo funciona el constructor de Persona.
  • Emplea Step Over para avanzar línea por línea en el bucle for y observar los cambios en las variables.
