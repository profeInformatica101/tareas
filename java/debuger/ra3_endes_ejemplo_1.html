Tareas para Probar el Debugger

Código Java para las tareas 
package com.ejemplos.debugger;

public class Ejemplo1 {
    public static void main(String[] args) {
        int[] numeros = {10, 20, 30, 40, 50};
        int suma = 0;
       
        // Suma de los elementos del arreglo
        for (int i = 0; i <= numeros.length; i++) { // Error: Índice fuera de rango
            suma += numeros[i];
        }

        // Calcula el promedio
        double promedio = suma / numeros.length;

        System.out.println("Suma: " + suma);
        System.out.println("Promedio: " + promedio);
    }
}
1. Identificar un error de ejecución con un breakpoint
  • Establece un breakpoint en la línea del bucle for.
  • Ejecuta el programa en modo depuración.
  • Observa el índice i en la ventana de variables y confirma que intenta acceder a un índice fuera del rango.
2. Explorar el flujo del bucle con Step Over
  • Usa Step Over para avanzar línea por línea dentro del bucle.
  • Observa cómo cambia el valor de suma en cada iteración.
  • Confirma que el bucle intenta acceder a un índice no válido al final.
3. Corregir el error y validar el cálculo del promedio
  • Corrige la condición del bucle reemplazando i <= numeros.length por i < numeros.length.
  • Vuelve a ejecutar el programa en modo depuración.
  • Verifica que el valor de suma y promedio sea correcto.
4. Evaluar expresiones durante la depuración
  • Establece un breakpoint en la línea donde se calcula el promedio.
  • Usa la funcionalidad Evaluate Expression para comprobar los valores de:
    • suma
    • numeros.length
    • suma / numeros.length

Objetivo General

Estas tareas enseñarán a los estudiantes:

  • Cómo identificar y solucionar errores de ejecución.
  • Cómo inspeccionar y modificar variables durante la depuración.
  • Cómo evaluar expresiones y comprender el flujo del programa.
