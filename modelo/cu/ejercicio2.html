Detalle del Caso de Uso: Registrar Historias Clínicas

Nombre del Caso de UsoRegistrar Historias Clínicas
ID del Caso de UsoUC1
PrecondiciónEl veterinario debe estar autenticado en el sistema.
PostcondiciónLa historia clínica del paciente está actualizada y almacenada en la base de datos.
DescripciónEste caso de uso permite a los veterinarios registrar las historias clínicas de los pacientes, incluyendo información como diagnósticos y tratamientos administrados durante cada visita.
Flujo Principal
  • El veterinario inicia sesión en el sistema.
  • El veterinario selecciona la opción de crear nueva historia clínica.
  • El veterinario completa los campos requeridos (datos del paciente, diagnóstico, tratamiento).
  • El veterinario envía la información para su almacenamiento.
  • El sistema confirma que la historia clínica ha sido registrada correctamente.
Caminos Alternativos
  • A1 (Paso 3): Si falta información esencial, el sistema muestra un mensaje de error y solicita los datos faltantes.
  • A2 (Paso 4): Si el sistema no puede almacenar la información, se notifica al veterinario y se solicita un nuevo intento.

Descripción de la Actividad

Descripción del Caso de Uso:
"Programar Recordatorios" es un caso de uso crucial que permite al personal administrativo y a los veterinarios configurar recordatorios para citas futuras, vacunaciones, revisiones de salud y otros tratamientos necesarios para las mascotas. Los recordatorios son esenciales para garantizar el cumplimiento del plan de tratamiento de cada mascota y para mantener informados a los propietarios sobre los cuidados necesarios.
